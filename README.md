# subscription-box 
An iOS app for a subscription box for exotic fruits

## About
An iOS app for a subscription box for exotic fruits, but can be used as a template for other subscription box based services.

### Features
* Login Screen
* Home Screen
* New Box Selection Screen
* History Screen
* User Profile Screen
